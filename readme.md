# 📋 Survey Form

A responsive and user-friendly Survey Form built using HTML, CSS, and JavaScript. This project allows users to submit survey responses through a clean and interactive interface.

## 🚀 Features

- Responsive design for all devices
- User-friendly interface
- Form validation using JavaScript
- Clean and modern UI
- Easy to customize and extend

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```text
survey-form/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 📸 Preview

Open the project in your browser to view the survey form and submit responses.

## 🔧 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/survey-form.git
```

2. Navigate to the project folder:

```bash
cd survey-form
```

3. Open `index.html` in your browser.

## 💡 Usage

- Enter the required information in the form.
- Complete the survey questions.
- Click the Submit button.
- JavaScript validates the input before submission.
